# Santa and his Penguins
This game is a holiday game about a Santa who had a few penguins to help him deliver christmas presents, but faces an obstacle the Grinch who can command infinite amounts of drones, because he buys in bulk. The grinch flies on his drone throwing pizza-pies and commanding drones agains Santa and his penguins.
